ALYS DB 002 JPN
Codename ALYS4AE JP. This is the first publicly released Japanese vocal library along with ALYS4AE FR.

This is a modified version of the ALYS DB 002 JPN voicebank that is configured to work in UTAU. It contains extra recordings taken from DB 001 JPN.
This is a monopitch VCV voicebank.

Recommended Resampler: moresampler
ALYS has very slow consonants, so a high consonant velocity will help her a lot.
